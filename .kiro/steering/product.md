# Product Overview

Ascenda is a self-growth and productivity platform focused on habit formation and long-term personal development. Unlike traditional to-do apps, it emphasizes consistency tracking, progress visualization, and user accountability.

## Core Purpose
Help users build consistent daily habits through goal tracking, progress visualization, and gamified engagement patterns inspired by successful apps like Duolingo.

## Key Features
- User authentication with secure JWT cookie-based sessions
- Daily goal creation and completion tracking
- Protected dashboard with progress visualization
- Streak tracking and habit formation metrics
- Future: AI-based goal suggestions, social features, mobile support

## Target Audience
Gen-Z users seeking gamified self-improvement tools with visual progress tracking and social accountability features.

## Current Status
MVP with authentication, basic goal management, and dashboard UI. Backend API complete, frontend dashboard uses placeholder data that needs to be connected to real endpoints.
